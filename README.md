# n8n-nodes-fluig

This is an n8n community node. It lets you use Fluig API in your n8n workflows.

O TOTVS Fluig Plataforma permite que sua empresa unifique processos, pessoas e sistemas com uma interface fácil de usar e intuitiva. Ele proporciona um ambiente de trabalho moderno, colaborativo, dinâmico e prático. Isso torna os processos do seu negócio muito mais ágeis e confiáveis. Sua equipe reduzirá o tempo com tarefas burocráticas, dedicando esforços no que gera mais lucro para o seu negócio.

[n8n](https://n8n.io/) is a [fair-code licensed](https://docs.n8n.io/reference/license/) workflow automation platform.
